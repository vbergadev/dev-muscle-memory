# Chapters 13–15 — Advanced (FP + Big-O)

From the book "Learning JavaScript Data Structures and Algorithms" (Loiane Groner).

## Exercises with tests (start here)
1. `01-compose-pipe.ts` — `compose` and `pipe`
2. `02-fibonacci-bigo.ts` — memoized fibonacci, O(n)

Stubs are in `src/` and the tests (failing on purpose) are in `tests/`. Make them pass.

## Then, on your own (no starter test)
- Big-O analysis of the previous exercises
- AVL / Red-Black tree (intuition)
- immutability and pure functions

## How to run
```bash
pnpm test
pnpm test:watch
```

Only consult the [author's repo](https://github.com/loiane/javascript-datastructures-algorithms) after trying on your own.
