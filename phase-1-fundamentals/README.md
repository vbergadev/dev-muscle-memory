# Phase 1 — Warm-up: JS + TS from scratch (weeks 1–5)

**Goal:** get back to writing JS/TS without relying on AI. Rebuild speed and muscle memory.

| Week | Folder | Topic |
|---|---|---|
| 01 | `week-01-js-fundamentals` | JS fundamentals revisited |
| 02 | `week-02-closures-and-this` | Event loop, closures, scope, `this` |
| 03 | `week-03-prototypes` | Prototypes, classes, inheritance |
| 04 | `week-04-typescript` | Essential TypeScript |
| 05 | `week-05-async-and-utils` | Async, Promises, classic utilities |

**Parallel DSA track:** chapters 1–6 of Loiane Groner's book, in `dsa/`.

**Phase checkpoint:** you type `useState`, `Promise.all`, `.map(x => ...)` without stopping to think. If not, extend this phase by 2 more weeks.
