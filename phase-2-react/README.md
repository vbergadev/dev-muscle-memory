# Phase 2 — React from basics to advanced (weeks 6–12)

**Goal:** rebuild React from JSX to advanced patterns, typing everything, without AI.

| Week | Folder | Topic |
|---|---|---|
| 06 | `week-06-react-basics` | JSX, components, props, state |
| 07 | `week-07-react-effects` | Effects, refs, mental lifecycle |
| 08 | `week-08-react-composition` | Composition, lifting state, controlled/uncontrolled |
| 09 | `week-09-react-hooks` | Context, useReducer, custom hooks |
| 10 | `week-10-react-performance` | Performance (memo/useMemo/useCallback) |
| 11 | `week-11-react-patterns` | Advanced patterns (compound, headless, slot) |
| 12 | `week-12-react-ecosystem` | TanStack Query/Router, Zustand |

Each week is a **Vite + React + TS** app. Tests use **Vitest + Testing Library** (jsdom environment).

**Parallel DSA track:** chapters 7–15 of Loiane Groner's book.

**Phase checkpoint:** you can build a non-trivial React component in 30 min without AI, with custom hooks, decent performance, and clear patterns. And you finished the book.
