# Phase 3 — Node from basics to advanced (weeks 13–18)

**Goal:** master the Node runtime, HTTP, API architecture, persistence, and testing.

| Week | Folder | Topic |
|---|---|---|
| 13 | `week-13-node-runtime` | Node runtime, modules, npm, CLI |
| 14 | `week-14-node-streams` | File system, streams, buffers |
| 15 | `week-15-express-api` | HTTP and Express/Fastify + Zod |
| 16 | `week-16-node-database` | Postgres + Drizzle (via Docker) |
| 17 | `week-17-node-auth` | Auth, JWT, security |
| 18 | `week-18-node-architecture` | Clean Architecture, testing, manual DI |

From week 16 on, start Postgres with `pnpm db:up` (Docker Compose at the root). Tests use **Vitest + Supertest** (node environment).

**Phase checkpoint:** you can build a Node API from scratch with auth, a database, tests, and decent architecture — typing it, without AI.
