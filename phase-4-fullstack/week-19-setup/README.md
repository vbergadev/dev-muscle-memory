# Week 19 — Definition and setup

## Goal of the week
Pick a medium-large project (Linear clone, mini Notion, real-time dashboard, habit tracker, Calendly-like scheduler) and set up the monorepo.

## What to do
- pnpm monorepo: `apps/web` (React+Vite+TS) + `apps/api` (Node+Express+TS) + `packages/shared` (common types).
- Postgres via Docker Compose (use the root one).
- CI/CD with GitHub Actions.
- ESLint + Prettier + Husky + lint-staged.
- Vitest + Playwright.

## Deliverable by end of week
Structured repo, professional README in English, first deploy (even hello world).

> **Phase 4:** AI may help with boilerplate — but only after you write the skeleton and the intent. See the root `CLAUDE.md`.
